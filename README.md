# Portfolio
## Project Overview

This is a fully responsive, single-page portfolio website designed with a bold neon/dark aesthetic. It includes the following sections:

- **Home** — Hero section with name, tagline, photo, and call-to-action buttons
- **Education** — Timeline of academic qualifications
- **Skills** — Technical languages, tools, and soft skills displayed as pill tags
- **Projects** — Cards showcasing real-world projects with GitHub links
- **Certifications** — Verified credentials from DataCamp, TCS iON, and MathWorks
- **Contact** — Full-width grid with email, phone, LinkedIn, Instagram, and GitHub links

## Setup Instructions

No server, build tool, or installation is required. This is a pure static website.

### To run locally:

1. **Download or clone the repository:**
   ```bash
   git clone https://github.com/rohithchowdary77/rohithchowdary77.github.io
   ```

2. **Navigate into the project folder:**
   ```bash
   cd rohithchowdary77.github.io
   ```

3. **Open the website:**
   - Simply double-click `index.html`, **or**
   - Open it in any modern browser (Chrome, Firefox, Edge, Safari)

> No npm install, no server, no dependencies — just open and view.

---

## How to Run the Project

### Option 1 — Open directly in browser
Double-click `index.html` from your file manager. It will open in your default browser instantly.

### Option 2 — Live Server (VS Code)
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. The site opens at `http://127.0.0.1:5500`

### Option 3 — GitHub Pages (Live Deployment)
The site is hosted live at:
```
https://rohithchowdary77.github.io
```

## Features

- Fully **responsive** — works on desktop, tablet, and mobile
- **Smooth scroll navigation** with active link highlighting
- **Scroll reveal animations** — sections animate in as you scroll
- **Mobile hamburger menu** — dropdown nav for small screens
- **Neon glow aesthetic** with animated background blobs
- **Photo card** with glowing corner bracket effect on hover
- **Project cards** with GitHub repo links
- **Certification cards** with issuer and badge tags
- **Contact grid** — all links open directly (email, phone, social)
